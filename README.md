# Triplex

An [apartment](https://github.com/influitive/apartment) for succesfull Phoenix
programmers.

## Installation

The package can be installed as:

  1. Add `triplex` to your list of dependencies in `mix.exs`:

    ```elixir
    def deps do
      [{:triplex, "~> 0.1.0"}]
    end
    ```

  2. Run in your shell:

    ```bash
    mix deps.get
    ```
